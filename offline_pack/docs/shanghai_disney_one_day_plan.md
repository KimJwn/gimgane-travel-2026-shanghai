# 상하이 디즈니랜드 1일 플랜 (, 4인 중년 가족) ## 고정 조건
- 방문일: - 인원: 4인
- 입장권: 오후권(얼리버드 사전구매)
- 출발: 푸동공항(PVG)
- 이동 1안: 디디 택시 약 45,000원
- 이동 2안: 지하철 `2(또는 10)호선 -> 환승 -> 11호선`, Disney Resort역 1번 출구
- 준비물: 양산모자(햇빛 대비)
- 숙소 거리: 디즈니 기준 약 5km ## 타임라인 (입력값 반영)
| 시간 | 할 일 | 메모 |
|---|---|---|
| 10:30 | 푸동공항 -> 디즈니 출발 | 디디(택시) 우선, 교통상황 따라 지하철 대체 |
| 12:00 | 디즈니 도착 목표 | 점심 먼저 or 입장 후 식사 선택 |
| 12:00~13:00 | 점심 + 짐 처리 | 캐리어는 `숙소 짐보관` 또는 `디즈니 무인짐보관함` |
| 13:00~18:00 | 어트랙션 집중 | 추천 루트: `트론 -> 주토피아 -> 캐리비안 해적` |
| 18:00~19:30 | 저녁(디즈니 내) | 앉아서 쉬는 식당 우선 |
| 19:30~21:00 | 쇼핑/산책/포토 | 체력 안배 구간 |
| 21:15 | 일루미네이션 시작 | 관람 여부 사전 결정 필요 |
| 21:40~22:30 | 디즈니 -> 숙소 이동 | 5km 구간, 디디 우선 | ## 핵심 의사결정: 21:15 일루미네이션 볼까?
### A안: 본다 (추천)
- 장점: 하루 하이라이트 완성, 재방문 가능성 낮으면 만족도 큼
- 단점: 귀가 동선 혼잡, 숙소 도착이 늦어짐
- 팁: 20:30 전후 관람 포지션 확보, 종료 직후 10~20분 천천히 이동 ### B안: 안 본다
- 장점: 피크 혼잡 회피, 귀가 편함
- 단점: 디즈니 핵심 야간 연출 미관람
- 팁: 20:40~21:00 사이 퇴장하면 이동 스트레스 감소 ## 실전 체크리스트
- 여권/티켓 QR/보조배터리 확인
- 입장 직후 앱으로 실시간 대기시간 확인
- 트론 대기 과도 시 순서를 `주토피아 -> 캐리비안 -> 트론`으로 유연하게 변경
- 저녁은 18시 전후로 당겨 혼잡 피하기 ## 캐리어 짐보관 상세
- 이용 가능: 입장 전 `Luggage Storage` 접수
- 절차: 접수 -> 보관증 수령 -> 당일 보관증으로 찾기
- 요금: 일반 캐리어 `80 RMB/개/일`, 소형 물품 `10 RMB/개/일`(3변 합 60cm 이하)
- 위치: `Wishing Star Park 서문 맞은편` 또는 `메인 입장 보안검색 동측` 표기
- 운영: 파크 폐장 후 최소 1시간 이상 운영(공식 안내 기준)
- 주의: 위험물/고가품/동식물 등 보관 불가, 여권/귀중품은 직접 소지 ## 무료 공연 + 위치 + 근처 어트랙션 ( 최적경로용)
- `주의`: 정확 시간은 당일 변동 가능. 아래는 최근 공식 편성 패턴.
- Mickey's Storybook Express(퍼레이드): 보통 낮 1~2회, `Gardens of Imagination`(중앙 허브)
- ILLUMINATE!: 보통 `21:15`, `Gardens of Imagination`(야간 메인)
- Avengers Training Initiative: 주간 다회, `Tomorrowland`(`TRON` 근접)
- Club Destin-E: 저녁 시간대 사례 다수, `Tomorrowland`(`TRON` 근접)
- Swabbie Fight / Eye of the Storm: 주간 다회, `Treasure Cove`(`캐리비안 해적` 근접)
- Mickey's Storybook Adventure: 주간 다회, `Mickey Avenue`(입출구/회복 동선)
- 동선 원칙: `Treasure Cove(캐리비안+공연) -> Tomorrowland(TRON+공연) -> 중앙허브(일루미네이션)` ## 기념품 샵 최적 타이밍
- 주토피아 구간: `Fashions by Fru Fru` (짧게 10~15분)
- 캐리비안 구간: `Doubloon Market` (근접 쇼핑)
- TRON 구간: `Intergalactic Imports/Power Supplies` (근접 쇼핑)
- 메인 구매: `Avenue M Arcade`(파크 내) 또는 `World of Disney`(Disneytown, 파크 외)
- 팁: 21:15 쇼를 볼 때 파크 내 샵 마감이 빠르면 `20:20~20:45`에 메인 구매 선실행 ## 예산 메모(사용자 입력 반영)
- 공항 -> 디즈니: 약 45,000원(디디)
- 오후 입장권: 얼리버드 사전구매 완료(금액만 추후 입력)
- 식비/간식/귀가 교통: 현장 확정 ## 지도 보기
